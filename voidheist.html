    About This Game

    üéØ The Objective

    Collect 3 Voidmatter cores and escape the space station before:

    • The AI reaches Alert Level 5
    • The Reactor melts down
    • You eliminate each other (very possible)

    ‚öôÔ∏è Core Mechanics

    1. Adaptive AI System

    The board reacts to players. Loud players trigger drone swarms. Hackers increase firewall difficulty. If someone hoards loot‚Ä¶ accidents happen. The AI evolves mid-game. It learns. Yes, Stargames wanted stress.

    2. Betrayal Protocol

    At setup, one card is secretly shuffled in: "Corporate Asset". If someone draws it, their real mission is to secure 2 Voidmatter cores and prevent others from escaping. They win alone. Paranoia guaranteed.

    3. Upgrade Economy

    Voidmatter can be used to: install cybernetic boosts, enhance weapons, override rooms, or sabotage others. Do you escape rich? Or escape alive? Choose wisely.

    üó∫ The Board

    The board is modular and changes every game. Hex-based spaceship tiles create dynamic layouts with rooms like:

    Reactor Core
    Data Vault
    Drone Bay
    Escape Hangar

    Tiles can be flipped to reveal hazards. Some rooms degrade over time. Yes, the ship is literally falling apart. No two games are the same.
